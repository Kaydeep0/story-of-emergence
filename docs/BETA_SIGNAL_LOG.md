## Classification Guide

**Category A — Locked-in**
- Returned without reminder
- Writing increased or stabilized
- Language indicates trust or calm
- Examples: "this feels quiet", "this feels respectful", questions about permanence/export

**Category B — Curious**
- Returned once
- Limited writing
- Neutral language
- No strong signals either direction

**Category C — Misaligned**
- Did not return
- Asked for guidance, prompts, or "what to do"
- Language indicates confusion about purpose
- Examples: "what should I write?", "how do I use this?", feature requests

**Rules:**
- One bucket only per user
- No debate
- No blending categories
